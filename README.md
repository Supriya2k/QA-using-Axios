# QA Automation Framework (Playwright + API)

## Overview
This project demonstrates a scalable QA automation framework using Playwright for UI and API testing.

## Tech Stack
- Playwright (UI + API testing)
- JavaScript (Node.js)
- GitHub Actions (CI/CD)

## Features
- UI test automation (login, cart, flows)
- API testing (GET, POST validation)
- Data-driven testing
- CI/CD pipeline

## Project Structure
tests/ui
tests/api
pages/
utils/

## How to Run
npm install
npx playwright test

## CI/CD
Tests run automatically on every push using GitHub Actions.
