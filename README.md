# QA-using-Axios

## Overview
## API Testing (Axios)

This project includes API testing using Axios:

- GET users endpoint
- POST login (success & failure)
- Error handling validation

Run API tests:
node tests/api/api.test.js

## Tech Stack
- Playwright (UI + API testing)
- JavaScript (Node.js)
- GitHub Actions (CI/CD)

## Features
- UI test automation (login, cart, flows)
- API testing (GET, POST validation)
- Data-driven testing
- CI/CD pipeline

## Project Structure
tests/ui
tests/api
pages/
utils/

## How to Run
npm install
npx playwright test

## CI/CD
Tests run automatically on every push using GitHub Actions.
